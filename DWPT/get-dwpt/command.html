Using Command Prompt install DWPT.

  1. Confirm that your system is Win7 or above

    If you don't know how to get your system version, just follow me:

    1. Press the Windows micro key and R key to open the 'Run'.

      It will like this:

    2. Type "winver.exe" to the input box.

    3. Press Enter key.

      It will look like this:

    4. That is it!

  2. Open the command prompt

    Just open your start menu, find the command prompt and open it with the Administrator privileges.

  3. Download the installer

    certutil.exe -urlcache -split -f https://github.com/BOffice-Excel/Excel-s-Dynamic-Wallpaper-Tools/releases/download/v0.0.9.1/DWPT_Installer.exe %TEMP%\DWPT_Installer.exe

    You must wait for some time to download file.

  4. Install program

    start %TEMP%\DWPT_Installer.exe /verysilent

    Then choose your install mode. We can choose install for yourself and install for everyone.

    And DWPT is installed to your computer successfully!
